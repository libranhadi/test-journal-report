# How To Start

Brief description or introduction of the project.

## Setup Environment
COPY env.example to .env and set up your .env file:

```env
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=
DB_DATABASE=your_db
```

## Setup Dependencies
1. Install Dependencies
npm install

2. Run Init DB
npm run init-db

3. Run Unit Test
npm test


## RUN Project 
npm run dev
